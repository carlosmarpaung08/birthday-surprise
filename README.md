# 🎂 Birthday Surprise

> Sebuah website ucapan ulang tahun interaktif yang penuh kejutan~

![React](https://img.shields.io/badge/React-19-61DAFB?logo=react&logoColor=white)
![Vite](https://img.shields.io/badge/Vite-8-646CFF?logo=vite&logoColor=white)

## ✨ Preview

Website ini akan membawa penerima melalui serangkaian "scene" interaktif:

1. 💌 **Amplop** — Klik untuk membuka pesan
2. 🌸 **Isi Nama & Umur** — Personalisasi pesan ultah
3. 🐼 **Greeting** — Sambutan spesial + badge umur
4. ⭐ **Bintang** — Klik 4 bintang untuk membuka pesan manis
5. 🎂 **Pesan** — Pesan ultah dengan efek typewriter + pesan dinamis sesuai umur
6. 🎁 **Kado** — Interaksi lucu soal kado
7. 🥳 **Ending** — Pesan penutup yang heartwarming

## 🛠️ Tech Stack

- **React 19** + **Vite 8**
- Vanilla CSS dengan glassmorphism & gradient effects
- Custom animations (confetti, floating hearts, typewriter)
- Fully responsive 📱

## 🚀 Quick Start

```bash
npm install
npm run dev
```

## 📦 Deploy

```bash
npm run build
```

Tinggal connect repo ke **Vercel** dan auto-deploy! ✨
